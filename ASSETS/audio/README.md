# Audio Assets

This folder holds browser-playable audio files for the `CATS64` prototype.

Current approach:

- use short placeholder sounds while gameplay systems are still moving
- replace them later with real audio files provided by the project owner
- keep names stable so the game code can swap assets without larger rewrites

Suggested naming examples:

- `level_up_placeholder.wav`
- `trip_oops.wav`
- `trip_yikes.wav`
- `feed_drop.wav`
- `trap_snap.wav`
