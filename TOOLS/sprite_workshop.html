CATS64 Tool

Sprite Workshop

Draw placeholder art for the browser prototype, export it as JSON, and keep the asset pipeline simple enough that we can wire sprites into gameplay quickly.

Recommended Size

24x24 is the default because it gives us enough room for top-down cats, the player, and readable state markers without getting chunky too fast.
