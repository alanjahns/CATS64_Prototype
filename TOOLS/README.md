# Tools

This directory is reserved for prototype-specific tools, including future art tools, exporters, debug helpers, and workflow utilities for `CATS64_Prototype`.

## Available Now

- `sprite_workshop.html` is a browser-based pixel editor for prototype sprites
- Exported JSON sprite data is intended to live under `../ASSETS/sprites/`
- Supported canvas sizes are `16x16`, `24x24`, and `32x32`
